What are your options?

Running a
Virtual Machine

  • Low risk
  • Easier to setup
  • Limited performance
  • Less features

Virtual machines are best for only limited usage of Linux

WSL can also be used as an additional option for those who are interested in running a virtual machine

Dual-boot installation

  • Best of both worlds
  • Full performance and features
  • More risk
  • Might require workarounds

Dual-booting is best for full dives into Linux with a moderate dependency on Windows

Recommended to keep Linux and Windows installations on separate drives

(Windows doesn't play nicely with Linux installs)

Full installation

  • r/linuxmasterrace
  • Can act smug around Windows users
  • Big commitment
